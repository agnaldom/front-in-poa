Saiba por que você não pode
perder o Front in POA!

Já colocamos Porto Alegre no mapa como um dos principais centros de desenvolvimento front-end do país. Além de organizar o BrazilJS, acreditamos que é primordial termos um evento que consagre designers e desenvolvedores de internet.

Sabemos que entregar um projeto para a web não é tarefa fácil e queremos te ajudar nisto. As palestras abordam desde concepção de produto, marcação, estilo, acessibilidade, usabilidade até performance.

Este evento é obrigatório para todo o profissional que se preocupa com a experiência do usuário. Seja você atendimento, publicitário, designer gráfico ou programador, temos a certeza de que você irá aproveitar e aprender algo novo.

    Como se tornar um "Cara Foda"

    É super comum você ouvir alguém dizer: fulano é foda, beltrano é foda. Mas quando tu vai falar com a pessoa considerada foda, ela normalmente não pensa assim. As vezes as pessoas tem medo de não serem modestas, e eu tentarei dismistificar isso. Essa é uma palestra de motivação, inspiração e empreendedorismo individual. Espero com ela fazer mais pessoas atingirem o status de "Cara Foda"

  • 09:50
    Thiago Alves
    Testando o seu CSS

    Alterar um código CSS é uma tarefa simples. Mas como acontece com os demais elementos envolvidos em um projeto web, criar rotinas para testes garante minimamente que novas alterações não danifiquem o que já estava funcionando. Por ser declarativo, o CSS complica a realização de testes. Contudo, existem métodos e ferramentas que nos auxiliam nesta tarefa de garantir a camada de estilo.

    Componentes CSS: como desenvolver pensando em reuso

    Um dos principais pilares para a web emplacar como uma plataforma para se construir aplicações robustas foi o CSS. Porém, cada vez mais precisamos de aplicações dinâmicas e modularizadas. E como criar componentes encapsulados em uma linguagem onde tudo é global? Nessa talk iremos ver paradigmas e aprender como desenvolver CSS escalável em tempos de módulos JavaScript, HTTP/2 e Web Components.
